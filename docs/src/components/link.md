## API

| 参数     | 描述                   | 类型      | 默认值 |
| :------- | :--------------------- | :-------- | :----- |
| newtab   | 链接是否在新标签也打开 | `boolean` | false  |
| href     | 链接的目标地址         | `string`  | '/'    |
| disabled | 是否禁用链接           | `boolean` | false  |
